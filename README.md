# Interactive AI

Hands-on visualizations for machine learning concepts. Navigate with mouse or hands-free using webcam gesture control.

**[ai.neevs.io](https://ai.neevs.io)**

## Content

| Topic | Modules |
|-------|---------|
| Neural Networks | Neuron anatomy, activations, loss landscapes, optimizers |
| Convolutional Networks | Convolution, kernels, pooling, architectures |
| Reinforcement Learning | MDPs, Q-learning, policy gradients, PPO |

## Gesture Control

Point to navigate, hold to select. Uses MediaPipe hand tracking via webcam.

## Development

```bash
make install
make dev
```
